Este proyecto *analiza los precios de productos alimenticios en MiPymes cubanas* y su relación con el salario promedio, con el objetivo de *evaluar el poder adquisitivo real del trabajador*.

A partir de datos de mercados, salarios y gramajes de consumo, se cuantifica cuánto representa cada producto (y una *canasta básica mínima*) en términos de ingreso mensual y días de trabajo necesarios.

El aumento sostenido de los precios en las MiPymes ha generado un debate constante sobre el acceso real a productos básicos en Cuba. Sin embargo, muchas discusiones se quedan en percepciones y no en datos.

**Este proyecto nace con la intención de:**
- Traducir precios a métricas comprensibles (salario y días de trabajo).
- Comparar mercados de forma objetiva.
- Aplicar herramientas de Ciencia de Datos a un problema económico cotidiano.

****La motivación principal es demostrar cómo el análisis de datos puede aportar claridad y evidencia a una realidad social compleja.****
